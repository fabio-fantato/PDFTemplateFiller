DejaVu Sans font for tests

This folder is intended to contain a permissively-licensed font used by the test-suite when running in headless CI environments.

Please download and place a copy of DejaVu Sans (DejaVuSans.ttf) here. Recommended sources:

- https://dejavu-fonts.github.io/
- https://www.fontsquirrel.com/fonts/dejavu-sans

License: DejaVu fonts are offered under the Bitstream Vera / DejaVu licensing which permits bundling in repositories for testing purposes. Verify license compatibility with your project before distributing.

After placing DejaVuSans.ttf in this folder, tests will automatically pick it up.
